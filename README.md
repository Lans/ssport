# Ssport

由于Shadowsocket时不时端口就被墙了。每次都要经过一下步骤`登录服务器` -> `修改配置文件` -> `重启SS服务` 来换端口号或者加密方式。虽然步骤不多，每次及密码修改重启这些命令和密码记忆起来也麻烦，毕竟都是不常用的命令。为节省大家时间，于是写了一个命令行工具，可以一个命令行解决所有，省去繁杂操作。

## Installation

    $ gem install ssport

## Usage


具体用法可以看： `ssport -h` 查看。
